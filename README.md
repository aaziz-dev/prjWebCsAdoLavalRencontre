# ❤️ Friendbook – Social Networking Web App

## 📌 Overview

Friendbook is a social networking web application that allows users to connect with others based on shared interests and personal criteria.
Users can create profiles, search for other members, and communicate through an internal messaging system.

---

## ⚙️ Tech Stack

* C# (ASP.NET Web Forms)
* ADO.NET (DataSet, DataReader, LINQ)
* SQL Server / MS Access
* HTML, CSS

---

## 🏗️ Architecture

This project follows a **client-server multi-layer architecture**:

* 🖥️ GUI Layer: Web interface (ASP.NET Web Forms)
* 💾 Data Layer: Database access using ADO.NET

---

## 🚀 Features

### 👤 User Management

* User registration with personal information
* Secure login system (username & password)

### 🔍 Smart Search System

Users can search for other members based on:

* Gender
* Age category
* Ethnic group
* Interests (love, friendship, others)

### 💬 Messaging System

* Internal messaging between users
* Basic communication system within the platform

---

## 🧠 Key Concepts

* Multi-layer architecture (GUI + Data)
* Database integration with ADO.NET
* User authentication & session handling
* Dynamic search filtering
